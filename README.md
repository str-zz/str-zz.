## **What is this?**
This project/webapp is designed to calculate a rough estimate of various dosages based on drug, patient information, dosing factor, and dosage frequency
It comes in handy for healthcare professionals by offering a "starting line" to dosage estimations. It can also be used as a checking tool by various healthcare professionals!

## **Prerequisites**
- Web browser
- Internet access
- Text editor

## **Installation**
1. Clone code from repository
2. Paste onto Text editor
3. Save the file with the name index.hml
   - file must end with the .hml extension
4. Double-click the index.hml file
